# README

基于stm32 standard peripheral libraries的主板按键读取与识别练习

## 核心功能

* 功能点 1：实现了DevEBox stm32f407zgt6基础板的k0和k1按键外部中断和串口USART1的中断功能；

* 功能点 2：借助面向对象宏定义，优化按键key驱动和应用层代码，配合前向声明规避头文件循环依赖

  

## 运行环境与依赖

* **操作系统**：Windows 11
* **依赖库**：stm32 standard peripheral libraries
* **硬件**：USB转串口 CH340  DevEBox STM32F407开发板
